Keyboard shortcuts

General

  • Ctrl+Q - close application
  • Ctrl+S - save changes
  • Ctrl+Z - revert changes
  • Ctrl+G - go to frame/time
  • Shift+Delete - delete object

Creating new annotation objects

  • Alt+R - create new rectangle
  • Alt+C - create new circle
  • Alt+P - create new point
  • Alt+N - create new non-visual

Creating new annotation values

  • Alt+G - create new global annotation value
  • Alt+L - create new local annotation value

Navigation

  • Space - play/pause video
  • Left - go to previous frame
  • Right - go to next frame
  • Ctrl+Right - seek one second forward
  • Ctrl+Left - seek one second backward
  • Ctrl+Up (with object selected) - go to nearest next annotation
  • Ctrl+Up - go to nearest next frame with annotation
  • Ctrl+Down - go to nearest previous frame with annotation
  • Ctrl+Down (with object selected) - go to nearest previous annotation

Volume control

  • Ctrl+Plus - increase volume
  • Ctrl+Minus - decrease volume
  • Ctrl+* - mute
